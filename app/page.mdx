# Composable stacked dialogs for shadcn/ui

<Demo />

Stacked dialog component inspired by Radix UI Dialog. Accessible. Customizable. Open Source. Pre-styled for shadcn/ui.

        <DialogStackNext>Next</DialogStackNext>
